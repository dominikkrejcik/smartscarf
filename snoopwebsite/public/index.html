What does actually SmartScraf do?

The Scarf

The Scarf its self, allows user to connect to any mobile device via Bluetooth.

Bone Conduction

The Bone conduction tecnhology in the scarf allows the user to hear their surroundings while they are using the scarf.This increases safety and awareness.

3D Audio

The Smart Scarf is equipped with a all platforms mobile application.The application allows users to make conferences calls between people.By placing each user in 3d space, you fell that you are in the same room!!!

BioIMEI

To be able to make the SmartScarf amazing so snoop we have joined up with our friends from the BioIMEI team.The technology alows users to tap their phone with the Scarf and all you information will be loaded!!!
